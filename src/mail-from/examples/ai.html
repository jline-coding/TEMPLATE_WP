Mailform Pro

もっと究極の進化を遂げたメールフォームプロの登場です。

ChatGPT API またはDeepseek APIを利用してフォームから送信された内容を元にAI用プロンプトを生成し、生成されたテキストをメールに記載する機能です。

  1. config.cgiを編集し、push @Modules,'ChatGPT';またはpush @Modules,'Deepseek';を有効にします。
  2. ChatGPTの場合はOpen AI PlatformからAPIキーを取得します。Deepseekの場合はDeepseek PlatformからAPIキーを取得します。いずれのAPIも利用は有料になります。
  3. configs/ChatGPT.cgiまたはconfigs/Deepseek.cgiにAPIキーを設定し、プロンプトを作成します。
  4. config.cgiを編集し、管理者宛メールまたは自動返信メールの本文部分に、AIからの応答を表示したい場所に_%%ChatGPT%%_または_%%Deepseek%%_という文字列を配置します。
  5. これで作業は完了です。送信テストを行い、プロンプトが正常に処理されているか確認してください。。
